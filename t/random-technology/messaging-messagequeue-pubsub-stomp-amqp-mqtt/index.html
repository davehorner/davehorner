    MQTT || MQ Telemetry Transport

    Pervasive messaging
    MQTT - MQTT is a machine-to-machine (M2M)/"Internet of Things" connectivity protocol. It was designed as an extremely lightweight publish/subscribe messaging transport. It is useful for connections with remote locations where a small code footprint is required and/or network bandwidth is at a premium. tcp/ip 1883 or ssl 8883.
    MQ Telemetry Transport » wiki » mqtt_on_the_android_platform
    Using MQTT in Android mobile applications « dale lane - dirkmoors/MqttService - Mqtt service for Android, based on Eclipse Paho Mqtt Client and Dale Lane's blog article

    Mosquitto: An Open Source MQTT v3.1 Broker - MQTT v3.1 broker providing lightweight method of carrying out messaging using a publish/subscribe model.
